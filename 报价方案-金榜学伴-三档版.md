# 玉林金榜教育辅助服务有限公司小程序（暂定名：金榜学伴）报价方案（2000/4000/6000 三档）

版本：V1.2（对客沟通稿）  
日期：2026-01-21  

## 1. 交付目标
依据《玉林金榜教育辅助服务有限公司小程序需求规格说明书》，我们建议按“先试用、再上线、再扩展”的方式交付：先完成可演示、可试用的学习闭环（题库/课程/我的 + SVIP 权限），在确认方向后再逐步追加“付费/订单/运营后台/题目导入”等能力，避免首期投入过高。

## 2. 首期范围（聚焦可用）
- 小程序导航：首页 / 题库 / 资讯 / 我的
- 题库：初中/单招的目录浏览与题目预览（按年级/科目/单元组织）
- 课程：视频列表与播放页（首期以素材/链接方式展示）
- 我的：SVIP 码绑定入口与基础信息/反馈入口
- 权限：未绑定/未开通时提示开通（首期优先满足机构学生使用）

## 3. 报价（固定三档）
价格单位：人民币（含开发、联调、基础测试、上线协助）；不含第三方服务费、平台资源费与支付手续费。

### A 档：展示版（￥ 2,000）
适用场景：用于甲方确认页面风格、信息架构与基本交互流程。
- 品牌与文案统一（替换旧名称/LOGO/底部版权）
- 启动封面与首页基础布局优化
- Tab 架构调整：首页 / 题库 / 资讯 / 我的（可正常切换）
- 页面路由打通（可进入/返回；使用示例数据展示效果）

不包含：真实数据后台、题目批量导入、在线做题判分、进度/记录沉淀、SVIP/付费权限生效、订单功能。

### B 档：试用版（￥ 4,000）
适用场景：用于小范围试用，验证“找内容—看内容”的主流程。
- 含 A 档全部内容
- 资讯/公告：列表 + 详情（示例数据或静态配置）
- 题库浏览：学段（初中/单招）→ 年级/科目/单元 → 题目列表/详情（预览）
- 课程视频：视频列表 + 播放页（支持外链/存储链接播放）
- 我的：SVIP 绑定入口、基础信息与反馈入口（前端流程）

不包含：在线做题判分逻辑、观看/做题记录沉淀、SVIP 真校验与权限生效、付费会员/订单、运营后台与导入。

### C 档：上线准备版（￥ 6,000）
适用场景：具备“权限控制 + 核心学习闭环”的上线前形态（首期上线推荐）。
- 含 B 档全部内容
- SVIP 码：绑定校验、绑定状态展示（对接托管后端接口）
- 权限拦截：未绑定用户访问视频/试题/做题入口时提示开通（先按 SVIP 权限落地）
- 学习记录（基础）：记录最近观看与最近浏览（用于“继续学习”入口）

不包含：付费会员/微信支付/订单管理、运营后台完整功能、题目批量导入、OCR 识别、数据统计看板（均可二期增项）。

## 4. 二期增项（面议）
- 付费会员/微信支付/订单/发票
- 运营后台（轮播/资讯/题库/题目/SVIP/套餐/订单/导入）
- 题目导入（Excel/Zip/OCR 多格式）
- 数据统计与运营看板

## 5. 上线与审核支持

- 协助类目选择与审核材料准备

- 提交审核协助（含 1–2 次驳回修改）

1) 小程序 AppID（并将我方加入开发者）
