# 04-admin-console（管理台）

> 目标：当现有后台不可用时，提供自建管理台；可后置，优先保证导入与小程序可用。

## 范围与关键词

- 范围：题库管理、导入任务管理、用户/权限管理、内容管理
- 关键词：`admin`, `console`, `dashboard`, `rbac`, `cms`

## MVP 页面清单（对应 `TASKS.md`）

- 登录（管理员）
- 题库管理：科目 / 题库包 / 章节 / 题目列表
- 导入管理：上传 / 任务列表 / 任务详情 / 错误报告下载

## Web 端（开发占位）

- `web/`：Vite + React + TS 的最小骨架（登录 + 权限守卫占位）

## 路由与权限草案

- `MVP_ROUTES.md`：页面路由与最小权限点（等待 99-hub 定稿后对齐）

## 接口清单草案
