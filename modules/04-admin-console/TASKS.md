# 04-admin-console 任务看板

> 目标：当现有后台不可用时，提供自建管理台；可后置，优先保证导入与小程序可用。

## 0.1 本模块负责范围（关键词）
- `admin`, `resource-management`, `svip`, `banner`, `course`, `question-bank`, `orders`

## 0. 约束
- [ ] 仅修改 `modules/04-admin-console/**` 内文件
- [ ] UI 与权限/接口契约按 99-hub 定稿执行

## 1. MVP（可选/后置）
- [x] 登录与权限（管理员）
- [x] 题库管理：科目/题库包/章节/题目列表
- [x] 导入管理：上传、任务列表、任务详情、错误报告下载

## 2. 运营功能（后续）
- [x] 资讯/公告管理
- [x] 用户与权益管理（VIP/激活码）
- [x] 数据看板（活跃、正确率、完成率）

## 3. 需求补充（来自《需求规格说明书》）
- [ ] 轮播图管理：上传/排序/跳转链接/轮播开关/间隔
- [ ] 课程视频管理：学段/年级/科目/单元关联，封面/标题/简介/时长
- [ ] 试题管理：按学段/科目/单元/题型归档，支持批量导入/导出
- [ ] SVIP 码管理：批量生成/导出/有效期/绑定记录/注销
- [ ] 订单/套餐管理：套餐配置、订单列表、发票申请（后置但预留）

## 3.1 首页/资讯/商业配置（与小程序对齐）
> Tab 结构：`首页 / 题库 / 资讯 / 我的`
- [ ] 首页运营位配置：banner、推荐题库、推荐课程、推荐套餐（配置驱动）
- [ ] 资讯内容管理：列表/详情/上下线/置顶
- [ ] 套餐管理：价格、有效期、权益范围（先骨架）
