# 展示版（可直接运行）

本仓库提供一个用于“给甲方演示 UI/路由/基础流程”的展示分支：

- 分支：`demo/showcase`
- 标签：`demo-showcase-v1`

## 使用方式（微信开发者工具）
1. 切换到展示分支：
   - `git checkout demo/showcase`
2. 打开微信开发者工具 → 导入项目：
   - 项目目录：`modules/01-miniapp/miniprogram`
3. 运行后在底部 Tab 可看到：
   - 首页 / 题库 / 资讯 / 我的

说明：
- 展示版默认使用 Mock 数据，不依赖后端接口。
- 启动封面页“立即进入”可直接进入首页，不强制获取用户信息授权。
